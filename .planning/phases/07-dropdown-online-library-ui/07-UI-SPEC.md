# Phase 7 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Shadow DOM isolation, inline styles in Portal) |
| Preset | not applicable |
| Component library | none (React + inline CSS in getDropdownStyles()) |
| Icon library | lucide-react (already in use: FolderOpen, Sparkle, Brush, Layers, etc.) |
| Font | 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif |

**Note:** Content script dropdown uses Portal rendering to document.body with inline styles scoped by `#${PORTAL_ID}`. No Tailwind or shadcn in this context.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Dropdown gap from trigger, scrollbar width, menu margin-top |
| sm | 8px | Sidebar-item gap, header-actions gap, card gap, input field gap |
| md | 12px | Sidebar padding, category-item padding, header padding, content padding, item gap |
| lg | 16px | Header padding horizontal, content padding horizontal, card padding |
| xl | 24px | Empty-state padding, button sizes (24x24px) |
| 2xl | 32px | Trigger button size (32x32px) |
| 3xl | 48px | Reserved for major section breaks |

Exceptions: None. All values follow 4-point grid.

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Keyboard navigation | All cards and buttons tabbable, Enter/Space triggers action |
| Focus management | Modal traps focus, returns focus to trigger on close |
| Screen reader | aria-label on all interactive elements, aria-expanded on modal trigger |
| Color contrast | All text meets WCAG AA (4.5:1 for body, 3:1 for large text) |
| Motion | Fade animation respects prefers-reduced-motion (instant if set) |
