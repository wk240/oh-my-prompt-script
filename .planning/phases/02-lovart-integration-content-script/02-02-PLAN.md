# Phase 02: Lovart Integration - Plan 02

## Objective

Create Shadow DOM UI infrastructure and TriggerButton component. This plan establishes the isolated UI container and the trigger button that users click to open the dropdown.

## Context

- @`.planning/phases/02-lovart-integration-content-script/02-UI-SPEC.md` - Component specs, spacing, color
- @`.planning/phases/02-lovart-integration-content-script/02-RESEARCH.md` - Shadow DOM React pattern
- @`src/content/input-detector.ts` (from Plan 02-01) - Input positioning reference
- @`src/content/style-extractor.ts` (from Plan 02-01) - LovartStyleConfig
- @`src/shared/constants.ts` - EXTENSION_NAME

## Tasks

### Task 1: Implement UIInjector Class with Shadow DOM

**Action:**
1. Create `src/content/ui-injector.ts`
2. Implement `inject(inputElement: HTMLElement)` method:
   - Create host element: `<div id="lovart-injector-host">`
   - Attach Shadow DOM: `attachShadow({ mode: 'open' })`
   - Inject `<style>` tag with CSS (per UI-SPEC spacing/color)
   - Create React mount point: `<div id="react-root">`
3. Implement `positionHost(inputElement)` method:
   - `position: absolute`
   - Position left of input per D-01: `left: ${inputRect.left - 44 - 8}px` (button + gap)
   - `zIndex: 9999`
4. Add reposition handlers for scroll/resize events
5. Implement `remove()` cleanup method

**Implements:** EXT-02 (Shadow DOM isolation), D-01 (positioning)
**Verification:**
- Shadow DOM created on inject call
- Host element positioned correctly relative to mock input
- Style tag contains UI-SPEC CSS values
- Cleanup removes host element

### Task 2: Create TriggerButton Component

**Action:**
1. Create `src/content/components/TriggerButton.tsx`
2. Implement React component with props:
   ```typescript
   interface TriggerButtonProps {
     isOpen: boolean
     onClick: () => void
     lovartIconColor: string
   }
   ```
3. Lightning bolt SVG icon (per D-03):
   ```tsx
   <svg viewBox="0 0 24 24" fill="currentColor">
     <path d="M13 2L3 14h9l-1 8 10-12h-9l1-8z"/>
   </svg>
   ```
4. Apply styles per UI-SPEC:
   - Size: `44px x 44px` (WCAG touch target)
   - Border radius: `8px`
   - Background: Lovart button color (runtime extracted)
   - Icon color: `lovartIconColor` prop (per D-04)
5. Hover/active states: `#e8e8e8`, `#dcdcdc`
6. Accessibility: `role="button"`, `tabindex="0"`, `aria-label="插入预设提示词"`

**Implements:** CORE-01, D-02 (minimal icon), D-03 (lightning), D-04 (Lovart color)
**Verification:**
- Component renders with lightning icon
- Click handler triggers callback
- Hover/active states visible in browser
- ARIA attributes present

### Task 3: Create DropdownApp Root Component

**Action:**
1. Create `src/content/components/DropdownApp.tsx`
2. Implement root component coordinating:
   - State: `isOpen`, `selectedPromptId`
   - TriggerButton + DropdownContainer composition
3. Wire LovartStyleExtractor for icon color
4. Implement toggle behavior: click TriggerButton toggles `isOpen` (per D-12)
5. Pass `SAMPLE_PROMPTS` and `SAMPLE_CATEGORIES` to dropdown
6. Mount in Shadow DOM via UIInjector

**Implements:** D-11 (keep open logic prep), D-12 (toggle behavior)
**Verification:**
- App renders in Shadow DOM
- Click trigger toggles dropdown visibility
- State management functional
