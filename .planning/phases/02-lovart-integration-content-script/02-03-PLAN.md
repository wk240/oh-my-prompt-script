# Phase 02: Lovart Integration - Plan 03

## Objective

Build dropdown menu components with Lovart-native styling. This plan creates the dropdown container, category headers, prompt items with preview, and empty state UI.

## Context

- @`.planning/phases/02-lovart-integration-content-script/02-UI-SPEC.md` - Component inventory, typography, interaction states
- @`src/content/components/DropdownApp.tsx` (from Plan 02-02) - Parent component
- @`src/content/sample-data.ts` (from Plan 02-01) - SAMPLE_PROMPTS, SAMPLE_CATEGORIES
- @`src/shared/types.ts` - Prompt, Category interfaces

## Tasks

### Task 1: Implement DropdownContainer Component

**Action:**
1. Create `src/content/components/DropdownContainer.tsx`
2. Implement with props:
   ```typescript
   interface DropdownContainerProps {
     prompts: Prompt[]
     categories: Category[]
     onSelect: (prompt: Prompt) => void
     isOpen: boolean
     selectedPromptId: string | null
   }
   ```
3. Apply styles per UI-SPEC:
   - Position: `absolute`, `top: 48px` (below trigger)
   - Width: `280px`
   - Max height: `320px` with `overflow-y: auto`
   - Background: `#ffffff`
   - Border: `1px solid #e0e0e0`
   - Border radius: `8px`
   - Shadow: `0 4px 12px rgba(0,0,0,0.15)`
   - Padding: `12px`
4. Implement open/close animation per UI-SPEC:
   - Open: `opacity 0→1, translateY -4px→0, 150ms ease-out`
   - Close: `opacity 1→0, 100ms ease-in`
5. Reset scroll position to top on open

**Implements:** CORE-02, EXT-03, D-08 (Lovart-native styling)
**Verification:**
- Dropdown renders when `isOpen=true`
- Animation transitions smooth
- Scroll behavior functional
- Lovart-native visual style applied

### Task 2: Implement CategoryHeader and PromptItem Components

**Action:**
1. Create `src/content/components/CategoryHeader.tsx`:
   ```typescript
   interface CategoryHeaderProps {
     category: Category
     itemCount: number
   }
   ```
   - Font: `12px`, `500`, `#999`, uppercase
   - Padding: `8px 12px`
   - Margin: `12px` above (first: 0)

2. Create `src/content/components/PromptItem.tsx`:
   ```typescript
   interface PromptItemProps {
     prompt: Prompt
     isSelected: boolean
     onClick: () => void
   }
   ```
   - PromptName: `14px`, `500`, `#333`
   - PromptPreview: `12px`, `#666`, `max-width: 250px`, ellipsis
   - Preview length: `50 chars` (per D-06)
   - Padding: `12px`
   - Gap: `8px` (name to preview)
   - Border radius: `8px`
   - Hover: `#f8f8f8`
   - Selected: `#e6f4ff` + `2px solid #1890ff` left border

**Implements:** D-05 (name+preview), D-06 (50 char), D-07 (category grouping)
**Verification:**
- CategoryHeader renders with correct styling
- PromptItem shows name and truncated preview
- Hover/selected states visible
- Click triggers onSelect callback
