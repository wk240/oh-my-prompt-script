<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Popup UI → Service Worker | chrome.runtime.sendMessage crosses context boundary |
| Service Worker → chrome.storage.local | Storage API boundary |
| Console output | Logs may be observed by developer tools |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-02 | Information Disclosure | Console logs in service-worker.ts | mitigate | Log baseUrl and modelName only, apiKey omitted or masked |
| T-10-03 | Information Disclosure | SettingsApp.tsx console logs | mitigate | Use apiKey: '[REDACTED]' in any debug logs |
| T-10-04 | Tampering | API config input validation | mitigate | URL must start http/https, fields non-empty trim |
| T-10-05 | Information Disclosure | chrome.storage.sync usage | accept | Not used — explicitly using chrome.storage.local per AUTH-02 |
</threat_model>
