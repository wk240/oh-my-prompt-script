## Changes Made

### Task 1: Add 404/403/500 error handlers to classifyApiError()

**File:** `src/lib/vision-api.ts`

- Added handler for **404 (endpoint not found)**:
  - type: 'network'
  - message: 'API 端点不存在，请检查 Base URL 配置'
  - action: 'reconfigure'

- Added handler for **403 (forbidden)**:
  - type: 'invalid_key'
  - message: 'API 访问被拒绝，请检查 API Key 权限'
  - action: 'reconfigure'

- Added handler for **500/502/503 (server errors)**:
  - type: 'network'
  - message: 'API 服务暂时不可用，请稍后重试'
  - action: 'retry' (respecting MAX_RETRY_COUNT)

- Added console.error logging for unhandled errors to aid debugging
